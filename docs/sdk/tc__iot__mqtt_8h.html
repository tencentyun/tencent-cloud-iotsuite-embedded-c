enum _tc_iot_mqtt_client_state_e

客户端连接状态

枚举值
CLIENT_INVALID 

非法状态，数据未初始化

CLIENT_INTIALIAZED 

已初始化，连接相关信息已生成

CLIENT_NETWORK_READY 

网络层(TCP 或 TLS)连接已完成，可进行 MQTT CONN 请求

CLIENT_CONNECTED 

应用层连接完成：MQTT CONN/CONNACK 成功

enum _tc_iot_mqtt_conn_ack_e

MQTT 连接请求回包，CONNACK 返回码 returnCode

枚举值
TC_IOT_CONN_SUCCESS 

连接成功

TC_IOT_CONN_PROTOCOL_UNACCEPTABLE 

连接失败，不支持请求的协议版本

TC_IOT_CONN_CLIENT_ID_INVALID 

连接失败，Client Id 判定非法

TC_IOT_CONN_SERVICE_UNAVAILABLE 

连接失败，后端服务不可用

TC_IOT_CONN_BAD_USER_OR_PASSWORD 

连接失败，用户名或密码校验不通过

TC_IOT_CONN_NOT_AUTHORIZED 

连接失败，用户访问未授权
